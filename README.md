\# Plain / Flat Area Detector using Automatic SRTM DEM



This tool detects flat / plain areas inside any KML overlay by

automatically downloading SRTM DEM from OpenTopography.



\## Features

✔ Upload KML polygon  

✔ Auto-download 30m SRTM DEM  

✔ Extract plain / low slope areas  

✔ Export as flat\_areas.kml  

✔ Works in Streamlit Cloud  



\## Run locally

pip install -r requirements.txt

streamlit run app.py



\## Deploy to Streamlit Cloud

Upload this repo → click "Deploy"
